<¿Cómo funciona @supports?>

La regla @supports se usa para escribir bloques de código CSS que solo se aplicarán si el navegador soporta ciertas propiedades o valores. Si el navegador reconoce la propiedad que estás probando, aplicará los estilos dentro del bloque; si no, los ignorará.

<¿Por qué es útil @supports?>

Mejora la compatibilidad: Permite adaptar el diseño según las capacidades del navegador, ofreciendo una mejor experiencia para todos los usuarios.

Facilita la transición a nuevas tecnologías: Puedes probar nuevas propiedades y asegurarte de que no afecten a navegadores que aún no las soportan.

Mantiene el código limpio y organizado: Evita la sobrecarga de hacks o scripts para comprobar compatibilidad.
